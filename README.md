## 🎯 Working Philosophy

### **Real-World Impact Focus**

실제 사용자에게 가치를 전달하는 것을 최우선으로 합니다. 빠른 MVP 제작과 실사용자 피드백을 통해 제품을 진화시키는 **Lean Startup** 방식을 지향합니다.

### **Documentation & Knowledge Management**

모든 개발 과정을 문서화하여 **재현 가능한 운영**을 만들어갑니다. CONTEXT → README → CLAUDE 흐름으로 지식을 축적하고 재사용합니다.

### **Risk Management**

장애 대응과 성능 개선을 위해 **모니터링 체계**를 먼저 구축합니다. 배포 전후 체크리스트를 운영하여 안정적인 서비스를 제공합니다.

### **Technical Depth**

- **ORM First**: 모든 DB 접근은 ORM을 통해, Raw Query는 성능 검증 후만 사용
- **Pattern Driven**: Repository, Unit of Work, Domain Service 패턴 적용
- **Test First**: 테스트 커버리지 80%+ 목표, 회귀 방지
